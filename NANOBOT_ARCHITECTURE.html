一、核心组件列表

1. Agent（智能体）

Agent 名称 文件位置 职责描述
主 Agent agent/loop.py 接收用户消息，调用 LLM，执行工具，发送响应
子 Agent agent/subagent.py 后台任务执行，隔离工具集，有限迭代（最多 15 次）

2. Skills（技能库）

技能名称 文件位置 描述
SKILL-WEATHER skills/ 获取天气预报（内置技能示例）
SKILL-CALCULATOR skills/ 执行数学计算（内置技能示例）
用户自定义技能 workspace/skills/SKILL*.md 用户自定义技能，支持依赖检查

3. Tools（工具集）

工具名称 文件位置 描述
read_fileagent/tools/filesystem.py读取文件内容
write_fileagent/tools/filesystem.py写入文件内容
execagent/tools/exec.py执行 shell 命令
web_searchagent/tools/web.py网络搜索
messageagent/tools/message.py发送/等待消息（渠道通信）
spawnagent/tools/spawn.py生成子 Agent 任务
cronagent/tools/cron.py定时任务管理
subagentagent/subagent.py子 Agent 管理工具

4. Providers（LLM 提供商）

提供商 文件位置 描述
LiteLLMProviderproviders/litellm_provider.py通用提供商，支持 20+ LLM 服务
AzureOpenAIProviderproviders/azure_openai_provider.py直接 Azure OpenAI API（绕过 LiteLLM）
CustomProviderproviders/custom_provider.py自定义 API 端点
OpenAICodexProviderproviders/openai_codex_provider.pyOpenAI Codex 专用

5. Services（服务）

服务名称 文件位置 描述
MessageBusbus/queue.py异步消息队列（inbound/outbound）
SessionManagersession/manager.py会话管理（JSONL 持久化）
CronServicecron/service.py定时任务调度
HeartbeatServiceheartbeat/service.py心跳任务（30 分钟检查）
ChannelManagerchannels/manager.py渠道管理（Telegram、WhatsApp 等）

二、各 Agent 的 Skills 和 Tools

主 Agent (agent/loop.py)

可用 Skills:
- 工作目录 skills/SKILL*.md
- 内置技能 nanobot/skills/SKILL*.md
- always=true 的技能自动加载到 system prompt

可用 Tools:
- read_file   : 读取文件内容
- write_file  : 写入文件内容
- exec        : 执行 shell 命令（带超时和危险命令检查）
- web_search  : 网络搜索
- message     : 发送/等待消息（渠道通信）
- spawn       : 生成子 Agent 任务
- cron        : 定时任务管理（add/list/remove/enable/run）

子 Agent (agent/subagent.py)

可用 Skills:
- 通过主 Agent 传递的 system prompt 继承

可用 Tools:
- read_file   : 读取文件内容
- write_file  : 写入文件内容
- exec        : 执行 shell 命令
- web_search  : 网络搜索
（无 message/spawn 工具，防止无限递归）

八、子 Agent 启动和管理

8.1 SubagentManager 架构

class SubagentManager:
    def spawn(self, task: str, label: str | None = None) -> str:
        """生成子 Agent 任务"""
        task_id = str(uuid.uuid4())[:8]
        bg_task = asyncio.create_task(
            self._run_subagent(task_id, task, label, origin)
        )
        self._running_tasks[task_id] = bg_task
        return f"Subagent [{label}] started (id: {task_id})"

    async def _run_subagent(self, task_id, task, label, origin):
        # 1. 创建隔离工具集
        tools = ToolRegistry()
        tools.register(ReadFileTool, WriteFileTool, ExecTool, WebSearchTool)

        # 2. 运行有限迭代循环（最多 15 次）
        iteration = 0
        while iteration < 15:
            response = await provider.chat_with_retry(...)
            if response.has_tool_calls:
                # 执行工具
            else:
                final_result = response.content
                break

        # 3. 宣布结果
        await self._announce_result(task_id, label, task, final_result, origin)

11.3 Cron 表达式说明

格式：分 时 日 月 周 │ │ │ │ │ │ │ │ │ └─ 星期 (0-6, 0=周日) │ │ │ └──── 月份 (1-12) │ │ └─────── 日期 (1-31) │ └────────── 小时 (0-23) └───────────── 分钟 (0-59) 示例: - "0 9 * * *" → 每天 9:00 - "0 9 * * 1-5" → 每周一至周五 9:00 - "0 */2 * * *" → 每 2 小时 - "30 8 1 * *" → 每月 1 号 8:30
